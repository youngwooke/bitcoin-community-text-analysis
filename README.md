# 비트코인 커뮤니티 텍스트 분석

## 📌 개요
본 프로젝트는 비트코인 관련 온라인 커뮤니티 텍스트를 분석하여 투자 정보의 신뢰성을 평가하는 것을 목적으로 한다.  
자연어처리 기법을 활용하여 커뮤니티 담론이 객관적 정보 중심인지, 감정적 반응 중심인지 분석하였다.

---

## 🧠 분석 방법

### 1. 데이터 수집
- 수집 대상: DCInside 비트코인 갤러리
- 게시글 수: 4,301개
- 수집 도구: Octoparse

### 2. 전처리
- 특수문자 제거 및 공백 정규화
- 제목 + 본문 결합
- 형태소 분석기 **Okt** 사용
- 명사 추출 및 불용어 제거

---

## 🔍 분석 내용

### 📊 1. 단어 빈도 분석
- 상위 20개 명사 추출
- 투자 관련 단어와 감정 표현이 동시에 나타남

### 🤖 2. 의미 분석 (Transformer)
- 모델: Sentence-BERT (한국어)
- 주요 단어 간 의미 관계 분석
  - 매수, 매도, 거래소, 보유

### 📚 3. 군집 분석
- 문장 임베딩 기반 군집화 수행
- 담론 유형 분류:
  - 정보 중심
  - 투자 판단
  - 감정 표현
  - 잡담/노이즈

### 💬 4. 감정 분석
- 모델: KLUE-BERT 기반 감정 분석 모델
- 결과:
  - 중립: 67.4%
  - 부정: 24.9%
  - 긍정: 7.7%

---

## 📈 주요 결과

- 커뮤니티 담론은 투자 정보와 감정 표현이 혼재된 구조
- 감정적 표현 및 주관적 의견 비중이 높음
- 중립 표현이 많지만, 실제 담론은 감정 중심으로 형성됨
- 따라서 커뮤니티 정보는 투자 판단에 활용하기에 한계 존재
